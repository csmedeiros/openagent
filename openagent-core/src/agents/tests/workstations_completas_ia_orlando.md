## Considerações Importantes para Compra

### Requisitos de Energia
- RTX 4090: Fonte de 850W+ recomendada
- RTX A6000: Fonte de 1000W+ recomendada
- Dual GPU: Fonte de 1600W+ com eficiência 80+ Titanium

### Refrigeração
- Workstations com RTX 4090 requerem refrigeração líquida ou sistema de refrigeração avançada
- Verificar espaço no local de instalação (algumas workstations são muito grandes)
- Considerar ruído operacional para ambientes de escritório

### Software e Suporte
- Verificar se o sistema vem com CUDA Toolkit pré-instalado
- Frameworks populares (TensorFlow, PyTorch) devem estar otimizados
- Garantia e suporte técnico especializado em IA

### Prazos de Entrega
- Workstations customizadas: 2-4 semanas
- Modelos padrão: 1-2 semanas
- Sob encomenda enterprise: 4-8 semanas

## Dicas para Compra em Orlando

1. **Verificar disponibilidade local** - Muitas workstations de alto desempenho precisam ser encomendadas
2. **Considerar opções online** - Maior variedade e frequentemente melhores preços
3. **Negociar com revendedores** - Especialmente para compras corporativas
4. **Verificar programas de leasing** - Para empresas, pode ser mais vantajoso
5. **Garantia estendida** - Importante para investimentos de alto valor

Para adquirir uma workstation completa com GPU de 20GB+ VRAM em Orlando, as melhores opções são:

1. **Para orçamento limitado:** System integrators como Exxact Corp ou Lambda Labs com RTX 4090
